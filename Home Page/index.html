PILIHAN MAGANG

Junior Data Scientist (Pengolahan Data)

Junior Data Scientist adalah posisi pemula dalam bidang data science yang bertugas membantu menganalisis data dan membangun model sederhana untuk mendukung pengambilan keputusan bisnis.

Junior Data Analyst

Junior Data Analyst bertanggung jawab untuk memproses, membersihkan, dan memvisualisasikan data agar dapat digunakan dalam pembuatan keputusan strategis.

Junior Software Engineer

Junior Software Engineer membantu dalam proses pengembangan aplikasi dengan menulis, menguji, dan memperbaiki kode sesuai kebutuhan proyek.

Junior UI/UX Designer

Junior UI/UX Designer bertugas mendesain tampilan antarmuka aplikasi yang menarik dan mudah digunakan untuk meningkatkan pengalaman pengguna.
